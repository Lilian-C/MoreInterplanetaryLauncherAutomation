# InterPlanetary Launcher Automation

A mod for **Oxygen Not Included** that adds automation logic ports to the Interplanetary Launcher.

## Features

- **Ready to Launch** — Sends a green signal when the launcher has enough radbolts to launch a payload
- **Radbolts Empty** — Sends a green signal when the radbolt storage is empty (useful for refilling automation)

## Building

1. Open `InterPlanetaryLauncherAutomation.sln` in Visual Studio
2. Build in Release mode
3. Run `build-and-deploy.ps1` to compile and copy the mod to your game's mod folder

## Requirements

- Oxygen Not Included (Spaced Out! DLC recommended)
- .NET Framework 4.7.1
